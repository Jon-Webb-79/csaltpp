SIMD Utilities

This section documents the SIMD trait templates and operation specializations used to accelerate matrix operations for supported data types (float and double). These utilities are primarily used internally by matrix classes but may be of general interest to advanced users.

SIMD Traits

template<typename T>
class simd_traits

A generic trait structure to determine SIMD support for a given type.

Template Parameters:

T – The data type to query for SIMD support

static constexpr bool supported

Indicates whether SIMD is supported for the given type.

static constexpr std::size_t width

Number of elements processed per SIMD register.

template<>
class simd_traits<float>

Specialization of simd_traits for float. Uses compile-time flags to determine SIMD capability.

Var supported:

true if SIMD_WIDTH_FLOAT > 1

Var width:

SIMD vector width in elements

template<>
class simd_traits<double>

Specialization of simd_traits for double.

Var supported:

true if SIMD_WIDTH_DOUBLE > 1

Var width:

SIMD vector width in elements

SIMD Operations

template<typename T>
class simd_ops

Base template class for SIMD operations on type T. Specialized for supported types.

template<>
class simd_ops<float>

Provides AVX/SSE/fallback implementations of basic arithmetic operations on float arrays.

static void add(const float *a, const float *b, float *result, std::size_t size)

Performs element-wise addition of two float arrays.

static void sub(const float *a, const float *b, float *result, std::size_t size)

Performs element-wise subtraction of two float arrays.

static void add_scalar(const float *a, float scalar, float *result, std::size_t size)

Adds a scalar to each element of a float array.

static void sub_scalar(const float *a, float scalar, float *result, std::size_t size)

Subtracts a scalar from each element of a float array.

static void mul(const float *a, const float *b, float *result, std::size_t size)

Performs element-wise multiplication of two float arrays.

static void mul_scalar(const float *a, float scalar, float *result, std::size_t size)

Multiplies each float array element by a scalar.

static void div_scalar(const float *a, float scalar, float *result, std::size_t size)

Divides each float array element by a scalar.
