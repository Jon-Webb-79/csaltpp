Detailed Description

template<typename T>
class slt::MatrixBase< T >

Abstract base class for matrix types.

Provides a uniform interface for different matrix implementations (e.g., dense, sparse), supporting essential matrix operations such as element access, mutation, and cloning.

Template Parameters
TThe numeric type of the matrix elements (e.g., float, double).
