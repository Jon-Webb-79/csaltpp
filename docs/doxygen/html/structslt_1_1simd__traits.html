Detailed Description

template<typename T>
struct slt::simd_traits< T >

SIMD trait template to determine SIMD capabilities for a given type.

This generic template assumes no SIMD support. Specializations for specific types (e.g., float, double) provide actual SIMD support info.

Template Parameters
TThe data type to query for SIMD support.
